python: struct模块处理二进制数据

发表于 2017-01-04 | 更新于 2018-09-11 | 阅读次数 |

struct模块中最重要的三个函数是pack(), unpack(), calcsize()

#pack(fmt, v1, v2, …)
按照给定的格式(fmt)，把数据封装成字符串(实际上是类似于c结构体的字节流)

#unpack(fmt, string)
按照给定的格式(fmt)解析字节流string，返回解析出来的tuple

#calcsize(fmt)
计算给定的格式(fmt)占用多少字节的内存

struct中支持的格式如下表：
